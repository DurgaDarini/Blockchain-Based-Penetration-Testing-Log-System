# Blockchain-Based-Penetration-Testing-Log-System
This project presents a blockchain-based solution for securely storing penetration testing findings.
The system ensures tamper-proof vulnerability logging using Ethereum smart contracts.

## key Features
- Secure vulnerability logging
- Smart contract based storage
- Role-based access (Admin, Tester, Auditor)
- Blockchain-based transparency

## Technology Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Solidity
- Ethereum
- Hardhat
- Ethers.js
